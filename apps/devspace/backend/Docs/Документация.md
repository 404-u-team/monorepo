# Документация по бэкенду DevSpace

## Переменные окружения (в скобочках значения по-умолчанию)

**API_PORT** - порт, который слушает REST API (8080)

**DB_HOST** - хост бд postgres (localhost)

**DB_PORT** - порт, на котором работает БД postgres (5432)

**DB_USER** - пользователь postgres под которым должны исполняться запросы (postgres)

**DB_PASSWORD** - пароль от пользователя БД (postgres)

**DB_NAME** - имя самой БД (postgres)

**DB_SSLMODE** - использовать ssl или нет (disable)

**GORM_LOGGER** - включить журналер от GORM или нет (false)

**ARGON_MEMORY_USAGE** - максимум памяти под argon2id в КБ (54)

**ARGON_ITERATIONS** - итерации при создании хэша (5)

**ARGON_PARALL** - кол-во потоков при хэшировании (4)

**ARGON_SALT_LEN** - длинна соли (16)

**ARGON_KEY_LEN** - итоговая длинна хэша (32)
